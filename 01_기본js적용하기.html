자바스크립트 적용하기

  1. inline 방식 : 태그의 이벤트 속성에 javascript 코드를 작성하는 방식
  2. internal 방식 : script태그 내부에 필요한 javascript코드를 작성하는 방식
  3. external 방식 : js파일을 javascript코드를 작성하고 script태그로 불러와서 적용하는 방식

inline방식으로 적용하기

internal방식으로 적용하기
