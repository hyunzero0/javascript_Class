비교논리연산자

== : 일치여부를 확인하는 것. String도 적용 가능(자바 -> equals, 모든 게 객체라서 == 비교 가능)
!= : 불일치여부를 확인하는 것.
* 두 연산대상의 자료형이 다를 때 자료형을 자동으로 변환해서 비교함

=== : 일치여부를 확인하는데 자료형까지 확인(String "100", Number 100 는 다른 객체)
!== : 불일치여부를 확인하는데 자료형까지 확인
> : 수가 크다
< : 수가 작다
<= : 수가 작거나 같다
>= : 수가 크거나 같다
